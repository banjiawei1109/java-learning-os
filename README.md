# Java Learning OS

> 🚀 一个面向 Java 后端开发的 AI 学习操作系统。

## 📖 项目简介

Java Learning OS 是我为自己设计的一套学习系统。

目标不是简单地记录笔记，而是通过 AI + Git + 项目实践，完成 100 天 Java 后端学习计划，最终获得 Java 后端开发 Offer。

整个学习过程都会记录在这个仓库中，包括学习路线、每日记录、项目开发、面试准备以及成长轨迹。
