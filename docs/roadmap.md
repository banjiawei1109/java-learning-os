# Java Learning OS Roadmap

## 🎯 总目标

100 天内掌握 Java 后端开发核心技能，完成两个项目，并开始投递 Java 后端岗位。

---

## 第一阶段（Day 1 ~ Day 20）

目标：夯实 Java 基础

学习内容：

- Java 基础语法
- 面向对象
- 集合框架
- 异常
- IO
- 多线程
- Lambda
- Stream

输出：

- 每日学习记录
- Java 笔记
- 小练习

---

## 第二阶段（Day 21 ~ Day 45）

目标：Spring Boot

学习内容：

- Maven
- Spring
- Spring Boot
- MyBatis Plus
- MySQL
- REST API

输出：

- 第一个 Spring Boot 项目

---

## 第三阶段（Day 46 ~ Day 70）

目标：企业开发能力

学习内容：

- Redis
- Linux
- Docker
- Git
- Nginx
- JWT
- Swagger

输出：

- 项目功能完善

---

## 第四阶段（Day 71 ~ Day 100）

目标：求职

学习内容：

- JVM
- 八股文
- 项目优化
- 简历
- 模拟面试

输出：

- 简历
- GitHub 项目
- 开始投递
