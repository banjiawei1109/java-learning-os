## 📚 今日学习

### 1. Java 程序基本结构
```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, Java Learning OS!");
    }
}
```

关键概念：
- `public class`：类声明（文件名必须与类名一致）
- `main` 方法：程序入口
- `System.out.println()`：控制台输出

### 2. 开发环境
- Java 版本：Java 21（LTS）✅ 已安装
- IDE：IntelliJ IDEA
- 构建工具：Maven（后续使用）

### 3. 变量与数据类型
| 类型 | 说明 | 示例 |
|------|------|------|
| `int` | 整数 | `int age = 25;` |
| `double` | 浮点数 | `double price = 99.9;` |
| `boolean` | 布尔 | `boolean flag = true;` |
| `String` | 字符串 | `String name = "Java";` |
| `var` | 类型推断（Java 10+） | `var x = 10;` |

### 4. 练习代码与运行结果

**HelloWorld.java**
```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, Java Learning OS!");
        
        // 变量与数据类型练习
        int age = 25;
        double price = 99.9;
        boolean isStudent = true;
        String name = "Java学习者";
        
        System.out.println("姓名：" + name);
        System.out.println("年龄：" + age);
        System.out.println("价格：" + price);
        System.out.println("是否学生：" + isStudent);
    }
}
```

运行结果：
```
Hello, Java Learning OS!
姓名：Java学习者
年龄：25
价格：99.9
是否学生：true
```
