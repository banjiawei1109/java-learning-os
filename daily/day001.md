## 📚 今日学习

### 1. Java 程序基本结构
```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, Java Learning OS!");
    }
}
```

关键概念：
- `public class`：类声明（文件名必须与类名一致）
- `main` 方法：程序入口
- `System.out.println()`：控制台输出

### 2. 开发环境
- Java 版本：Java 21（LTS）
- IDE：IntelliJ IDEA
- 构建工具：Maven（后续使用）

### 3. 变量与数据类型
| 类型 | 说明 | 示例 |
|------|------|------|
| `int` | 整数 | `int age = 25;` |
| `double` | 浮点数 | `double price = 99.9;` |
| `boolean` | 布尔 | `boolean flag = true;` |
| `String` | 字符串 | `String name = "Java";` |

## 💡 关键收获
- Java 21 可以直接用 `javac` 编译，用 `java` 运行
- 类名必须与文件名一致（public class）
- `main` 方法是程序入口，必须有 `static`
